LinkSmart®

Free, Open Source IoT
Platform

LinkSmart® helps you to build Internet of Things applications

  • It provides services that allow you to build up the required infrastructure, such as device abstraction, provisioning, data storage, pub/sub, visualization and much more!
  • It offers you advances features such as stream mining or online machine learning

What LinkSmart® is

  • Free and Open Source
  • Relying on widespread standards (OGC SensorThings, MQTT, REST, TLS, etc.)
  • Proven in various application scenarios
  • Deploy the right servies for your application (no need to deploy the whole platform for a simple IoT prototype)
